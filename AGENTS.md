1. Classroom & User Management

    Role-Based Portals: Dedicated interfaces for Teachers (content creation, grading, analytics) and Students (learning, practice, submissions).

    Secure Enrollment: Classroom creation with unique 6-character invite codes for student access.

    Multilingual Support: Deep integration with Indic languages (Hindi, Tamil, Telugu, etc.) via Sarvam AI for both text and voice.

2. Intelligent Content Pipeline

    Automated RAG (Retrieval-Augmented Generation): Uploaded PDFs, docs, and YouTube transcripts are automatically chunked and embedded into a vector database for AI querying.

    Private Student Notebooks: Students can upload personal notes that the AI uses to provide personalized tutoring without sharing that data with the rest of the class.

    Audio-Book Conversion: Automatic background conversion of course materials into multilingual audiobooks with chapter-based navigation.

3. AI Tutoring System

    Hybrid AI Tutor: A chat interface supporting both text and voice (STT/TTS). The AI answers questions based strictly on course materials to prevent hallucinations.

    Source Citations: Every AI-generated answer includes specific citations (filename and page number) to the source material.

4. Automated Academic Tools

    AI Rubric Grader: Automatically grades assignments based on teacher-defined rubrics. It provides a weighted score, detailed reasoning, and actionable feedback for every criterion.

    Quiz & Flashcard Generator: One-click generation of practice quizzes (MCQ, True/False, Short Answer) and flashcards from uploaded course content.

    Spaced Repetition: Flashcards utilize the SM-2 algorithm to optimize student memorization based on their performance.

5. Analytics & Feedback

    Teacher Dashboard: High-level views of class performance, submission rates, and grade trends.

    Knowledge Gap Analysis: AI clusters common student questions to identify specific topics where the class is struggling.

    Assignment Pre-check: Students can run a "pre-submission review" to see how the AI would grade their draft against the rubric before final submission.

6. System Utilities


    Global Search: Semantic and keyword search across all classroom materials.
